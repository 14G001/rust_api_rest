Contiene las operaciones que deberán tener los distintos tipos de bases de datos del aplicativo; logrando así el patrón de diseño Abstract Factory; forzando a que los distintos tipos de bases de datos se implementen correctamente. Las bases de datos contienen los datos de los clientes y sus saldos.
